💻 软件方案

3.1 发射端 (Arduino Nano) — 固件代码

1 开发环境：安装 Arduino IDE，选择开发板 "Arduino Nano"，处理器 "ATmega328P (Old Bootloader)"
2 安装库：项目 → 加载库 → 管理库 → 搜索安装 SoftwareSerial（IDE 自带，不需额外安装） 
// ==========================================
// 发射端 (TX) — Arduino Nano + HC-12
// 按键按下 → 发送 Page Up / Page Down 指令
// ==========================================

#include <SoftwareSerial.h>

// 引脚定义
#define HC12_TX    2    // HC-12 RX
#define HC12_RX    3    // HC-12 TX
#define BTN_UP     4    // 上一页按键
#define BTN_DOWN   5    // 下一页按键

SoftwareSerial hc12(HC12_TX, HC12_RX);

void setup() {
  pinMode(BTN_UP,   INPUT_PULLUP);
  pinMode(BTN_DOWN, INPUT_PULLUP);
  
  hc12.begin(9600);       // HC-12 默认波特率 9600
  delay(1000);
}

void loop() {
  static unsigned long lastSend = 0;
  unsigned long now = millis();
  
  // 防抖 + 去重 (200ms 间隔)
  if (now - lastSend > 200) {
    if (digitalRead(BTN_UP) == LOW) {
      hc12.println("UP");
      lastSend = now;
    }
    if (digitalRead(BTN_DOWN) == LOW) {
      hc12.println("DOWN");
      lastSend = now;
    }
  }
  
  // 低功耗优化（可选）：delay 替换为 sleep
  delay(20);
}

3.2 接收端 (Arduino Pro Micro) — 固件代码

1 开发环境：Arduino IDE，选择开发板 "Arduino Micro" 或 "SparkFun Pro Micro" (ATmega32U4, 5V/16MHz)
2 安装库：Keyboard.h (IDE 自带，ATmega32U4 原生支持) 
// ==========================================
// 接收端 (RX) — Arduino Pro Micro + HC-12
// 接收无线指令 → 模拟 USB 键盘 Page Up/Down
// ==========================================

#include <Keyboard.h>
#include <SoftwareSerial.h>

// 引脚定义
#define HC12_RX 2      // HC-12 TX → Pro Micro RX
#define HC12_TX 3      // HC-12 RX → Pro Micro TX
#define LED_PIN 4      // 状态指示灯

SoftwareSerial hc12(HC12_RX, HC12_TX);
String inputBuffer = "";

void setup() {
  pinMode(LED_PIN, OUTPUT);
  digitalWrite(LED_PIN, LOW);
  
  hc12.begin(9600);         // HC-12 默认波特率
  Keyboard.begin();           // 初始化 USB HID 键盘
  
  // 启动闪烁 3 次表示就绪
  for (int i = 0; i < 3; i++) {
    digitalWrite(LED_PIN, HIGH);
    delay(200);
    digitalWrite(LED_PIN, LOW);
    delay(200);
  }
}

void loop() {
  while (hc12.available()) {
    char c = hc12.read();
    if (c == '\n' || c == '\r') {
      // 收到完整指令
      inputBuffer.trim();
      
      if (inputBuffer == "UP") {
        digitalWrite(LED_PIN, HIGH);
        Keyboard.press(KEY_PAGE_UP);
        delay(50);
        Keyboard.releaseAll();
        digitalWrite(LED_PIN, LOW);
        
      } else if (inputBuffer == "DOWN") {
        digitalWrite(LED_PIN, HIGH);
        Keyboard.press(KEY_PAGE_DOWN);
        delay(50);
        Keyboard.releaseAll();
        digitalWrite(LED_PIN, LOW);
      }
      
      inputBuffer = "";
    } else {
      inputBuffer += c;
    }
  }
}

📝 完整开发步骤

Part A：发射端 (遥控器)

1 搭建 Arduino 开发环境
下载安装 Arduino IDE 2.x，连接 Arduino Nano，选择开发板和端口。
2 硬件接线
按照上方接线图，连接 HC-12 模块（VCC→5V, GND→GND, TX→D2, RX→D3）和两个按键（D4, D5 + GND）。注意 HC-12 的 SET 引脚悬空（正常工作模式）。
3 配置 HC-12 模块（关键步骤）
将 HC-12 的 SET 引脚拉低(GND)，用串口工具发送 AT 指令确认通信。推荐设置： 
AT+FU3         // 模式 FU3 (极低功耗, 5000bps, 最远距离)
AT+P8          // 发射功率 8 (最大, +20dBm)
AT+C001        // 频道 001 (可改 001-127)
AT+B9600       // 波特率 9600 (与代码一致)
设置完后 SET 引脚悬空，重新上电。
4 烧录固件
将上方发射端代码复制到 Arduino IDE → 编译 → 上传到 Nano。
5 供电方案
18650 电池 (3.7V) → TP4056 充电保护板 → 升压模块 (可选, Nano 支持 5V 直接接 VIN) → Arduino Nano VIN。加一个电源开关方便关机省电。
6 测试
通电后，用串口监视器(9600bps)连 Nano，按下按键应看到输出 "UP" 或 "DOWN"。

Part B：接收端 (接收器)

1 硬件接线
Pro Micro (5V) 接 HC-12（5V→VCC, GND→GND, D2→TX, D3→RX）。LED 接 D4+220Ω→GND。注意：Pro Micro 的 USB 同时供电和通信。
2 配置 HC-12
与发射端的 HC-12 完全相同的参数 (FU3, P8, C001, B9600)。两边的频道、波特率必须一致！
3 烧录固件
选择开发板 "Arduino Micro"，上传接收端代码。⚠️ 烧录时如果 HC-12 占用 Serial1 引脚导致上传失败，先拔掉 HC-12 的 TX/RX，上传成功后再接上。
4 插上电脑验证
LED 闪烁 3 次表示就绪。打开记事本或 Word，按下遥控器按键 → 应看到 Page Up / Page Down 效果。

🔧 关键技术要点

1. 为什么 Pro Micro 能模拟键盘？

Arduino Pro Micro 使用 ATmega32U4 芯片，内置 USB 控制器，支持 USB HID (Human Interface Device) 协议。通过 Keyboard.h 库，可以像蓝牙键盘一样发送标准按键码。

2. HC-12 频率干扰如何避免？

  • 设置不同频道：AT+C001 ~ AT+C127
  • 同一频道 + 不同地址可以实现多对多通信
  • 如果附近有其他 433MHz 设备，换一个频道即可

3. 如何扩展长按功能？

在发射端代码中增加长按检测逻辑，收到 INPUT_PULLUP=LOW 时启动计时器，超时发送 "UP_HOLD"，接收端持续 Keyboard.press(KEY_PAGE_UP) 直到收到 "UP_RELEASE"。

4. 低功耗优化（延长遥控续航）

  • Nano 空载 ~20mA → 用 Arduino Pro Mini (3.3V) 省一半
  • 利用 HC-12 的 FU3 模式 (80μA 待机)
  • Arduino 进入 IDLE 睡眠模式，按键中断唤醒
  • 2000mAh 18650 理论续航可达数周

🛠️ 故障排查

问题原因解决方法
接收不到信号HC-12 频道/模式不匹配两边都重新 AT 配置为完全相同的参数
距离很近但有丢包天线未接好或电源噪声焊接弹簧天线到 HC-12 的 ANT 焊盘, 电源加电容滤波
Pro Micro 无法上传HC-12 占用串口烧录时先拔掉 HC-12 的 D2/D3 线
电脑识别不到键盘烧录后未复位拔掉 USB 重插，或按一下 Pro Micro 上的 RST
按键无反应INPUT_PULLUP 未正确接线按键一端接 I/O 口, 另一端接 GND (不要接 VCC)
遥控距离不够模式选错 / 功率太低检查 AT+FU3 和 AT+P8, 确保两边 HC-12 天线焊接好
